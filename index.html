Market Basket Analysis Project

Project Overview

The Market Basket Analysis project aims to uncover associations between products purchased together by customers. By applying association rule mining, the project helps businesses understand customer purchasing behavior and improve cross-selling and marketing strategies.

Objectives

To identify frequently bought item combinations and recommend product bundling strategies to improve sales and customer experience.

Tools & Technologies Used

  • Python
  • Pandas
  • MLxtend (Apriori & Association Rules)
  • Spyder IDE / VS Code

Project Workflow

  1. Data Loading: Imported the market basket transaction dataset.
  2. Data Preprocessing: Cleaned and converted transaction data into a suitable format for analysis.
  3. Frequent Itemset Mining: Applied the Apriori algorithm to identify frequent item combinations.
  4. Association Rule Mining: Extracted strong association rules using support, confidence, and lift metrics.
  5. Rule Filtering: Selected the most relevant rules based on confidence and lift thresholds.
  6. Output Generation: Displayed and saved frequent itemsets and strong rules in a CSV file.
  7. Marketing Strategies: Suggested product bundling and cross-selling strategies based on the findings.

Key Findings

  • Identified top product combinations frequently purchased together.
  • Confidence and lift metrics highlighted strong cross-selling opportunities.
  • Association rules can help retailers design effective promotions and shelf placements.

How to Use This Project

  1. Download all the project files from the links above.
  2. Run the Python script using Spyder or VS Code.
  3. Review the output CSV for frequent itemsets and strong association rules.
  4. Refer to the project report for detailed insights and marketing recommendations.

Conclusion

The Market Basket Analysis project successfully identified valuable product associations. Businesses can leverage these insights to create effective bundling strategies, enhance customer satisfaction, and increase revenue through cross-selling.
